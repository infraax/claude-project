Optimization

The TGCH Stack

Four compounding optimization layers that reduce cost and improve response quality on every run.

T
Token Cache
Deterministic stable prefix with cache_control: ephemeral. Every repeated context hits the cache instantly.
89% average cache hit rate
G
Grammar Encoding
Typed dispatch routes tasks to the optimal format: typed_pseudocode, dsl, toon, or codeact.
76% reduction in format ambiguity
C
Compression
LLMLingua local preprocessor + Protocol Document deduplication. Reusable content-addressed blocks eliminate token waste.
62% of repeated context deduplicated
H
Habit Learning
384-dim semantic memory via LanceDB. KNN retrieval surfaces the most relevant past context before each dispatch.
54% hit rate on semantic retrieval

Get Started

Install in 60 seconds

Three ways to install — npm is recommended for CLI usage, VS Code extension for IDE integration.

# Install globally
npm install -g claude-project

# Or GitHub Packages
npm install -g claude-project --registry https://npm.pkg.github.com

# Verify
claude-project --version
# → 5.2.0
# 1. Initialise a project
claude-project init "My Project" -d "AI research pipeline" -s "Planning"

# 2. Register MCP with Claude Code
claude-project inject

# 3. Install session hooks (auto-load memory on every session)
claude-project hooks install --global

# 4. Start background daemon (macOS launchd)
claude-project daemon install

# 5. Verify everything
claude-project status
// Add to ~/.mcp.json (auto-done by `claude-project inject`)
{
  "mcpServers": {
    "claude-project": {
      "command": "claude-project",
      "args": ["mcp"]
    }
  }
}

// Or run the Python MCP server directly
{
  "mcpServers": {
    "claude-project": {
      "command": "python3",
      "args": ["/path/to/claude-project/mcp/server.py"]
    }
  }
}
# Install from VS Code Marketplace
code --install-extension claudelab.claude-project

# Or search in VS Code Extensions panel:
claudelab.claude-project
